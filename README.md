# Ukiah Senior Center - Event Tickets

Purchase tickets for events at the Ukiah Senior Center.

## Getting Started

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- Zeffy (payment processing)

## Features

- Event ticket purchasing via Zeffy popup modals
- Responsive design optimized for seniors
- Accessible, WCAG-compliant interface
- Easy integration with Zeffy ticketing system

## Development

This site uses Yarn with the Plug'n'Play linker disabled for compatibility.
